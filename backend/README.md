# Backend

## Python

Virtual env was created using:

```shell
python3 -m venv .venv
source .venv/bin/activate
```

## Java

Java was installed using:

```shell
# https://sdkman.io/
sdk install java 19.0.1-librca
```

## Go

Go was installed by following the [Go setup guide](https://go.dev/dl/).
You should be able to use any version of Go above `1.24.6`
